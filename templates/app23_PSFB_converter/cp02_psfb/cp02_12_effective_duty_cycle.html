Generalized Analytical Model for the Effective Duty Cycle in PSFB Converters

The ideal PSFB duty cycle is:

\[ D_{\mathrm{ideal}} = \frac{V_o}{V_{in}} \cdot \frac{N_p}{N_s}. \]

Parasitic voltage drops reduce the effective duty cycle. The table below summarizes the key parasitic contributions (nominal values from calculation):

Parasitic voltage drops in the PSFB converter
No. Quantity Expression Value [V]
1 Leakage inductance \(\delta V_{L_k} = L_k I_o \frac{N_s}{N_p} F_{sw}\) {{ results.delta_V_Lk }}
2 Transformer resistance \(\delta V_{TR} = I_o \left( \frac{N_s}{N_p} R_{prim} + \frac{N_p}{N_s} R_{sec} \right)\) {{ results.delta_V_TR }}
3 MOSFET on-resistance \(\delta V_{R_{dson}} = I_o \left( \frac{N_s}{N_p} \cdot 2 R_{dson,PR} + \frac{N_p}{N_s} R_{dson,SR} \right)\) {{ results.delta_V_Rdson }}
4 Output inductor \(\delta V_o = \frac{I_o}{2} R_{L_o}\) {{ results.delta_V_Lo }}

Normalized parasitic coefficients are defined as:

\begin{align} \alpha_{L_k} &= \frac{\delta V_{L_k}}{V_{in}}, & \alpha_{TR} &= \frac{\delta V_{TR}}{V_{in}}, & \alpha_{R_{dson}} &= \frac{\delta V_{R_{dson}}}{V_{in}}, \nonumber \\ \beta_o &= \frac{\delta V_o}{V_o}, & \alpha_\Sigma &= \alpha_{L_k} + \alpha_{TR} + \alpha_{R_{dson}}. \end{align}

For the nominal design:

\[ \alpha_\Sigma = \frac{ {{ results.delta_V_Lk }} + {{ results.delta_V_TR }} + {{ results.delta_V_Rdson }} }{V_{in}} = {{results.alpha_Sigma}}, \quad \beta_o = \frac{ {{ results.delta_V_Lo }}}{V_o} = {{results.beta_o}}. \]

The effective duty cycle including parasitic effects is compactly expressed as:

\[ \boxed{ D_{\mathrm{eff}} = D_{\mathrm{ideal}} \frac{1 + \beta_o}{1 - \alpha_\Sigma}, \quad \Delta D = D_{\mathrm{eff}} - D_{\mathrm{ideal}} }\]

Numerical evaluation for the nominal operating point:

\[ D_{\mathrm{ideal}} = {{ results.D_ideal }}, \quad D_{\mathrm{eff}} = {{ results.D_eff }}, \quad \Delta D \approx {{ results.Delta_D }} \ ({{ results.Delta_D_percent }}\% \ increase). \]
Discussion:
  • All parasitic effects are unified in a single formula.
  • Duty-cycle deviation is explicitly quantified.
  • Enables closed-form sensitivity analysis with respect to parasitics.
  • Fully consistent with detailed analytical calculations.
